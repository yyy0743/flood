FLOOD SG
This project is based on foundational data such as Singapore's Digital Elevation Model (DEM), transportation network, residential areas, and building footprints. Combined with water network and canal information, it conducts spatial analysis of potential urban flood risks and presents the estimated flood-prone areas to the public through computational analysis.
What Tools and Data did we use?

01 Elevation & Base Map Data

  • DEM of Singapore for terrain and floodplain analysis
  • Building footprints and residential area data for spatial context
  • Transportation network to assess accessibility impacts
  • Water network & canal data for flood path visualization

02 Data Processing & Visualization

  • QGIS: Spatial data processing and flood risk modeling
  • Mapbox: Interactive map rendering and thematic transitions

03 Participatory Data Input

  • Ushahidi: Crowdsourced flood reporting and real-time updates via user-submitted data

04 Website Design & Interaction

  • Figma: User interface wireframes and layout design
  • Visual Studio Code: Code editing and webpage implementation support
